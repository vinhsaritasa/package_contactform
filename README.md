## Laravel 5.x & 6.x

Install the ```vinhvo/contactform``` package:

```bash
$ composer require vinhvo/contactform
```

Add the NotificationsApiServiceProvider in ``config/app.php``:

```php
'providers' => array(
    // ...
    vinhvo\contactform\ContactFormServiceProvider::class,
)
```
